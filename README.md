ExpressionEvaluator is an expression parser coded in C++. The purpose of this program is to allow simple arithmetic expressions in the form of strings to be evaluated. Expressions can be copy-pasted directly into the terminal and solved. The Shunting Yard Algorithm is not utilized. Instead, the program counts the number of operations based on the number of mathematical symbols within the inputted expression. Then, the program loops through the entire input using a for loop. The loop stops once it identifies an operator, then creates a Pair object. Pairs consist of two operands and an operator (called an op because operator is a reserved keyword). Once the pair has been identified, a method is called that finds the result of the pair. Then, the pair within the string is replaced with its final value. The process repeats itself until the input has been simplified. Multiplication and division is given precedence over addition and subtraction.
